# De Cycloon — Vakantiehuis / Vacation home

Static, multilingual marketing website for **De Cycloon**, a holiday rental
home located at *Koninginnelaan 53, 8660 De Panne (Belgium)*.

Published at **[de-cycloon.be](https://de-cycloon.be)** and hosted on
[Netlify](https://www.netlify.com/).

---

## Overview

The site presents the holiday home to prospective guests in four languages
(**Dutch / English / French / German**) and exposes a live availability
calendar that the owner can update from Microsoft Word.

| Page             | File              | Purpose                                                     |
| ---------------- | ----------------- | ----------------------------------------------------------- |
| Intro            | `index.html`      | Welcome page with multilingual description of the property. |
| Location         | `location.html`   | Embedded Google Map of the address.                         |
| Gallery          | `gallery.html`    | Photo gallery of the house, rooms and garage.               |
| Calendar         | `calendar.html`   | Availability calendar (current + next year, via iframes).   |
| Style elements   | `elements.html`   | Template style/component reference (HTML5 UP demo).         |

The site is built on the free **"Massively"** template by
[HTML5 UP](https://html5up.net) (CCA 3.0 license). Styling and scripts
are stock template assets.

## The calendar workflow

The availability calendar is the only "dynamic" part of the site. Each year
is a single HTML file (`2026.htm`, `2027.htm`, …) generated by **Microsoft
Word** and rendered inside an `<iframe>` on `calendar.html`.

Colour coding in the calendar:

| Colour      | Code   | Meaning                                                                                  |
| ----------- | ------ | ---------------------------------------------------------------------------------------- |
| 🟥 Red      | BEZET  | Definitively occupied / Occupé définitivement / Besetzt endgültig / Definitively booked |
| 🟧 Orange   | OPTIE  | Option taken — contact the owner before booking                                          |
| 🟩 Green    | VRIJ   | Free / Libre / Frei                                                                      |

To update a year, the owner runs (from the `__tools__/` folder):

```bat
UpdateKalender.bat 2026
```

which:

1. Downloads `2026.htm` from the FTP server (`GetFileFromFtp.bat`).
2. Makes a timestamped `.bak` copy in `_backup/`.
3. Opens the file in Word so a VBA macro (see `MacroVoorKalender.txt`)
   can colour the day cells.
4. Uploads the edited file back via FTP (`PutFileFromFtp.bat`).
5. Deletes the local working copy.

`GitHub.bat` is the newer Git-based variant: it authenticates the GitHub
CLI with `token.txt`, syncs the repository, and backs up the current year
into `_backup/` so changes can be committed and pushed.

A step-by-step Dutch manual for the owner lives in
`__tools__/Hoe de kalender van De Panne aanpassen.docx`.

## Local preview

The site is pure static HTML/CSS/JS — no build step is required. Any
static file server works, for example:

```bash
# from the repository root
python -m http.server 8080
# then browse to http://localhost:8080/
```

The calendar iframes load sibling `*.htm` files relative to the page,
so they need to be served over HTTP rather than opened as `file://`.

## Deployment

The repository is connected to **Netlify**; pushing to the default
branch publishes the site. No build command is configured — Netlify
serves the repo root as-is.

Custom domain: `de-cycloon.be`.
